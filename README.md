# 2nd Anniversary Site

A simple static site to celebrate a 2-year anniversary. Files:
- index.html
- styles.css
- script.js

How to use:
1. Open [index.html](index.html) in your browser (double-click or use `Live Server`).
2. Replace the images in the gallery with your own photos.
3. Add the song file: place a file named `favorite-girl.mp3` into the `assets` folder so the path is `assets/favorite-girl.mp3`.

Quick personalization:
- Edit the hero text in `index.html` (the `h1` and the `#typewriter` element).
